# 📖 Course Highlights

## 🖥️ Computer Organization & Architecture

This course introduced the internal organization of computer systems and how hardware components interact to execute programs efficiently.

### Reflection

Computer Organization & Architecture provided a deeper understanding of how computers function beyond software. Learning about processors, memory hierarchy, instruction execution, and computer performance helped connect programming knowledge with the underlying hardware. The laboratory sessions and project activities enhanced understanding of computer architecture and system operations.

---

## 📊 Probability & Statistical Data Analysis

This course introduced probability concepts and statistical methods for analyzing and interpreting data.

### Reflection

Probability & Statistical Data Analysis was one of the more mathematically focused courses during the semester. In addition to learning statistical concepts and probability distributions, the course project involved collecting responses from the public and analyzing real-world data. This experience provided practical exposure to data collection, interpretation, and decision-making using statistical techniques, which are highly relevant to future studies in data analytics and data engineering.

---

## 💻 Programming Technique II

This course expanded upon the programming concepts introduced in Programming Technique I by introducing more advanced programming techniques and problem-solving approaches.

### Reflection

Programming Technique II was significantly more challenging than Programming Technique I as it introduced more complex programming concepts and larger problem-solving tasks. Learning a different programming approach and developing more structured programs improved coding proficiency and logical thinking. The mini project provided valuable experience in applying programming knowledge to develop a complete application.

---

## 📋 System Analysis & Design

This course introduced system development methodologies, requirements analysis, and system modelling techniques.

### Reflection

System Analysis & Design provided valuable exposure to how information systems are planned, analysed, and designed before implementation. Through the KADA project, practical experience was gained in requirements gathering, system modelling, and documentation using various diagrams and analysis techniques. The course strengthened teamwork, communication, and problem-solving skills while demonstrating the importance of proper planning in software development projects.

# 📝 Semester Reflection

Year 1 Semester 2 further strengthened the computing foundations developed during the first semester while introducing more practical and analytical aspects of computer science. The combination of programming, computer architecture, statistics, and system analysis courses provided a balanced learning experience that connected theoretical knowledge with practical applications.

Projects such as the public survey analysis in Probability & Statistical Data Analysis and the KADA project in System Analysis & Design provided opportunities to apply classroom knowledge to real-world scenarios. These experiences improved technical abilities, analytical thinking, communication skills, and teamwork while preparing for more advanced subjects in software engineering, databases, analytics, and data engineering.

Overall, this semester marked an important step in developing both technical competency and professional skills required for future studies and industry-related projects.
