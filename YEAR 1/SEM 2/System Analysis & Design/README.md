## 🚀 Featured Project: KADA-eServe System

KADA-eServe is a web-based cooperative management system developed as part of the course project. The system was designed to digitalize and streamline the operations of Koperasi Kakitangan KADA by replacing manual processes with an integrated information system.

### Project Scope

The system supports several core cooperative functions, including:

* Member registration and management
* Savings and contribution records
* Loan application and approval processes
* User and administrator management
* Report generation and record keeping

### Activities Performed

Throughout the project, various system analysis and design activities were carried out, including:

* Requirements gathering and analysis
* Stakeholder identification
* Functional and non-functional requirement specification
* Use case modelling
* Activity diagram development
* Process workflow analysis
* User interface design
* Project documentation and presentation


## 🌱 Reflection

System Analysis & Design provided practical exposure to the planning and design phases of software development. Through the KADA-eServe project, valuable experience was gained in analysing business requirements and translating them into structured system models before implementation.

The project demonstrated the importance of understanding user needs, documenting requirements clearly, and designing systems systematically to ensure successful software development. In addition to technical knowledge, the project strengthened teamwork, communication, documentation, and presentation skills through collaboration with group members throughout the analysis and design process.
