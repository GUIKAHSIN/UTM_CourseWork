# 📊 SSCP1123 Probability & Statistical Data Analysis

## 📚 Overview

This course introduces probability concepts and statistical methods used to analyze, interpret, and make decisions based on data.

## 📝 Evidence

* Assignment
* Statistical Analysis Project

## 🌱 Reflection

Probability & Statistical Data Analysis strengthened mathematical and analytical thinking skills through the application of statistical concepts to real-world data. The project, which involved collecting responses from the public and performing statistical analysis, provided practical experience in data collection, interpretation, and reporting. The course also highlighted the importance of data-driven decision making, which is highly relevant to data engineering and analytics.
