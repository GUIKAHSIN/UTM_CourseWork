# 📖 Course Highlights

## 🔢 SECI1013 Discrete Structure

This course introduced mathematical concepts that form the theoretical foundation of computer science.

### Key Learning Areas

* Logic and proof techniques
* Sets, relations, and functions
* Counting principles
* Graph theory
* Recursion

### Skills Developed

* Analytical thinking
* Mathematical reasoning
* Problem solving

### Reflection

Discrete Structure was one of the more challenging courses during the semester because it focused heavily on mathematical concepts and logical reasoning rather than practical programming. Topics such as logic, relations, functions, and graph theory required a different approach to problem solving and encouraged critical thinking. Although some concepts were difficult initially, the course helped strengthen analytical skills and provided a deeper understanding of the mathematical foundations that support computer science.

---

## 💻 SECJ1013 Programming Technique I

This course introduced fundamental programming concepts using C++.

### Key Learning Areas

* Variables and data types
* Control structures
* Functions
* Arrays
* File handling

### Skills Developed

* Programming fundamentals
* Algorithmic thinking
* Debugging techniques

### Reflection

Programming Technique I was one of the most interesting and enjoyable courses during the semester. It was my first experience with programming, and learning C++ sparked a strong interest in software development and problem solving. Through coding exercises and assignments, I learned how to translate ideas into working programs while developing logical thinking and debugging skills. The course provided a solid programming foundation that continues to support many subsequent computing subjects.

### Key Learning Areas

---

## ⚡ SECR1013 Digital Logic

This course focused on the fundamental principles of digital systems and computer hardware.

### Key Learning Areas

* Number systems
* Boolean algebra
* Logic gates
* Combinational circuits
* Sequential circuits

### Skills Developed

* Logical reasoning
* Circuit analysis
* Hardware fundamentals

### Reflection

Digital Logic was one of the most content-intensive courses during the semester. Learning topics such as Boolean algebra, logic gates, combinational circuits, and sequential circuits provided insight into how computers process information at the hardware level. Understanding the functions of AND, OR, NOT, NAND, NOR, XOR, and XNOR gates was particularly interesting, as it demonstrated how complex digital systems can be built from simple logical operations. The course strengthened logical reasoning skills and deepened my appreciation of computer architecture.

# 📝 Semester Reflection

Year 1 Semester 1 marked the beginning of my academic journey in Computer Science and Data Engineering. The combination of programming, mathematics, digital systems, and information technology courses provided a balanced introduction to both theoretical and practical aspects of computing.

Among all the courses, Programming Technique I had the greatest impact by introducing programming and sparking a deeper interest in software development. At the same time, Discrete Structure and Digital Logic strengthened logical reasoning and analytical thinking, while Technology & Information System provided exposure to the broader technology industry and its real-world applications.

Overall, this semester established a strong academic foundation and prepared me for more advanced courses in software engineering, databases, analytics, cloud computing, and data engineering in the coming years.
