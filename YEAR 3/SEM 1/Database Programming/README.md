# 🗄️ SECP3523 Database Programming

## 📚 Overview

Database Programming focuses on advanced database concepts, database programming techniques, and database-driven application development. The course extends database knowledge through practical implementation and management of data-intensive systems.

## 🌱 Reflection

Database Programming strengthened my understanding of database implementation beyond basic SQL and database design. Through practical projects involving MongoDB and database applications, I gained experience in data modelling, querying, indexing, and aggregation techniques. The course reinforced the importance of efficient data management and provided valuable experience in developing database-driven applications, which are fundamental components of modern data systems.
