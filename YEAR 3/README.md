Developed an end-to-end cloud data engineering solution based on Recoverable Assets (RA) and Inventory Risk Management (IRM) business scenarios.

### Technologies

* Azure Data Factory
* Azure Data Lake Storage Gen2
* Azure Databricks
* Azure Synapse Analytics
* Power BI

### Key Contributions

* Implemented Medallion Architecture (Bronze, Silver, Gold)
* Developed semantic views using Synapse Serverless SQL
* Built analytical dashboards in Power BI
* Applied cloud-based ETL and data warehousing concepts

# 📝 Reflection

Year 3 represented the most specialized and rewarding stage of my academic journey so far. Unlike previous years that focused primarily on programming fundamentals and software development, this year emphasized the practical application of Data Engineering, Artificial Intelligence, Business Intelligence, and enterprise systems technologies.

Courses such as Business Intelligence, High Performance Data Processing, and Special Topic in Data Engineering provided valuable exposure to working with real datasets, cloud platforms, analytical workflows, and modern data processing technologies. At the same time, Artificial Intelligence introduced machine learning concepts, while Database Programming and Enterprise System Development & Modelling strengthened database and system design knowledge.

One of the most valuable experiences throughout Year 3 was working on industry-oriented projects such as the Azure Inventory Risk Management Pipeline, Business Intelligence dashboards, and the TVETMARA Student Attendance Management System. These projects required the integration of multiple technologies and demonstrated how software development, data management, analytics, and cloud computing work together to solve real-world problems.

Overall, Year 3 significantly strengthened my technical expertise, analytical thinking, teamwork, communication, and project management skills. The knowledge and experience gained throughout this year further reinforced my passion for Data Engineering and prepared me for internship opportunities and future professional roles in data engineering, analytics, and business intelligence.
