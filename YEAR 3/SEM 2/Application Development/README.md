## 🚀 Featured Project: TVETMARA Student Attendance Management System

TVETMARA Student Attendance Management System is a Flutter and Firebase-based application developed to improve attendance management, timetable organization, and student discipline monitoring.

### Project Repository

🔗 Repository: https://github.com/GUIKAHSIN/ad_project_mara

### Key Features

* Attendance management
* Timetable management
* Student discipline reporting
* Role-based access control
* Firebase Authentication
* Cloud Firestore integration

### Technologies Used

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* GitHub

## 🌱 Reflection

Application Development introduced modern mobile application development using Flutter and Firebase, which were new technologies compared to previous software development courses. Through the TVETMARA Student Attendance Management System project, practical experience was gained in designing user interfaces, integrating cloud databases, implementing authentication mechanisms, and developing a complete application solution.

The project strengthened problem-solving abilities, software development skills, teamwork, and project management capabilities. More importantly, it demonstrated how software solutions can be designed to address real organizational challenges while providing a valuable introduction to cross-platform application development.
