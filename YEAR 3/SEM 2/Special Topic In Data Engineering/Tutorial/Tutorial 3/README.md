# 🤖 Tutorial 3 - AI Algorithms (ANN & CNN)

## 📖 Overview

This tutorial explored machine learning and image classification through the implementation of Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN). The CIFAR-10 dataset was used to train and evaluate different deep learning models.

The tutorial provided practical exposure to neural network architectures and demonstrated how CNN models can achieve better performance for image-related tasks.

---

## 🛠 Technologies Used

* Python
* TensorFlow
* Keras
* Matplotlib
* CIFAR-10 Dataset

---

## 🔄 What Was Done

* Prepared and explored the CIFAR-10 image dataset
* Developed an Artificial Neural Network (ANN) model
* Developed a Convolutional Neural Network (CNN) model
* Compared ANN and CNN performance
* Enhanced the CNN model through architectural improvements and optimization techniques

---

## 💡 Reflection

This tutorial provided practical experience in deep learning and image classification. The comparison between ANN and CNN models clearly demonstrated why CNN is the preferred architecture for image-related tasks due to its ability to extract important visual features automatically.

Another valuable lesson was learning that model performance can be improved through various enhancement techniques such as adding additional layers, applying dropout regularization, and using data augmentation. These improvements resulted in higher accuracy and better classification performance.

Overall, the tutorial strengthened understanding of neural networks and provided a solid introduction to deep learning concepts commonly used in artificial intelligence applications.
