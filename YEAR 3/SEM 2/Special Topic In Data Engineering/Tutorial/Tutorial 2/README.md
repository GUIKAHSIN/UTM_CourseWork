# ⚡ Tutorial 2 - Apache Spark

## 📖 Overview

This tutorial introduced Apache Spark and its role in big data processing. The tutorial focused on performing data extraction, transformation, and analysis using Spark while exploring how distributed computing can improve the efficiency of processing large datasets.

Through this tutorial, practical experience was gained in working with Spark DataFrames, data transformations, and analytical operations commonly used in modern data engineering workflows.

---

## 🛠 Technologies Used

* Apache Spark
* PySpark
* Python

---

## 🔄 What Was Done

* Loaded and explored a large dataset in CSV format using Spark
* Performed data cleaning and preprocessing on the CSV data
* Converted the processed CSV dataset into Parquet format for more efficient storage and analysis
* Applied filtering, aggregation, and transformation operations using Spark DataFrames
* Generated analytical insights from the transformed dataset
* Created visualizations based on the processed data

---

## 💡 Reflection

This tutorial provided valuable exposure to distributed data processing and demonstrated how Apache Spark differs from traditional data processing tools. Unlike conventional approaches that process data on a single machine, Spark utilizes parallel processing to handle larger datasets more efficiently.

The tutorial also highlighted how ETL processes can be performed within the Spark environment while supporting scalable data analytics. Overall, the experience strengthened understanding of big data technologies and their importance in modern data engineering solutions.
