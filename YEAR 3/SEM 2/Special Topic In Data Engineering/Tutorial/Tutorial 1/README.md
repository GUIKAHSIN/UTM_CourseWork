# ☁️ Tutorial 1 - Microsoft Azure

## 📖 Overview

This tutorial introduced the fundamentals of cloud-based data engineering using Microsoft Azure. A complete end-to-end data pipeline was developed by integrating multiple Azure services to ingest, store, transform, analyze, and visualize data.

The implementation followed the Medallion Architecture approach, providing practical exposure to how modern organizations manage data within cloud environments.

---

## 🛠 Technologies Used

* Microsoft Azure
* Azure Data Factory
* Azure Data Lake Storage
* Azure Databricks
* Azure Synapse Analytics
* Power BI

---

## 🔄 What Was Done

* Ingested data using Azure Data Factory
* Stored data in Azure Data Lake Storage
* Applied Medallion Architecture (Bronze, Silver, Gold)
* Performed data transformation using Azure Databricks
* Created analytical views using Azure Synapse Analytics
* Developed dashboards and visualizations using Power BI

---

## 💡 Reflection

This tutorial provided the first opportunity to build a complete cloud-based data engineering pipeline. Prior experience mainly involved local development environments, while Azure demonstrated how different cloud services can be integrated into a scalable data platform.

One of the most valuable lessons learned was understanding the flow of data throughout the entire pipeline, from ingestion and storage to transformation and visualization. The tutorial also introduced several industry-standard Azure services that are commonly used in modern data engineering projects.

Overall, the tutorial strengthened understanding of cloud computing and provided practical experience in implementing an end-to-end data engineering workflow.
