# 📖 Course Highlights

## 📱 Application Development

This course focused on designing and developing complete software applications using modern development frameworks, tools, and software engineering practices.

### Reflection

Application Development provided hands-on experience in developing modern applications using Flutter and Firebase, which were completely new technologies compared to previous programming courses. Through the TVETMARA Student Attendance Management System project, practical experience was gained in user interface development, authentication, database integration, and system implementation. The project demonstrated how software development concepts can be applied to solve real-world problems while improving teamwork, communication, and project management skills.

---

## 📊 Business Intelligence

This course introduced business intelligence concepts, data preparation, dashboard development, and decision-support systems.

### Reflection

Business Intelligence provided valuable experience in transforming raw data into meaningful business insights. Using Alteryx for data cleaning, transformation, and workflow development, followed by Power BI for dashboard creation and visualization, demonstrated how organizations leverage data to support decision-making. The course strengthened analytical thinking and highlighted the importance of data-driven business strategies.

---

## ⚡ High Performance Data Processing

This course focused on techniques and technologies used for processing large-scale datasets efficiently.

### Reflection

High Performance Data Processing introduced concepts related to efficient data processing, scalability, and performance optimization. Through the web crawler project, experience was gained in collecting, processing, and managing large volumes of data while exploring different processing techniques and performance considerations. The course provided valuable insight into how modern systems handle large-scale datasets and computational workloads.

---

## ☁️ Special Topic in Data Engineering

This course explored contemporary technologies, architectures, and industry practices within the Data Engineering field.

### Reflection

Special Topic in Data Engineering consolidated many of the fundamental concepts learned throughout the Data Engineering programme. Topics related to data engineering workflows, cloud platforms, analytical pipelines, Apache Spark, and Azure technologies provided a clearer understanding of how modern data systems are designed and managed. The course strengthened my understanding of the complete data lifecycle and reinforced my interest in pursuing a career in Data Engineering.

# 📝 Semester Reflection

Year 3 Semester 2 represented the culmination of many concepts learned throughout the Data Engineering programme. The courses emphasized practical implementation through industry-oriented projects and real-world datasets, requiring the integration of software development, analytics, cloud technologies, and data engineering concepts.

The TVETMARA Student Attendance Management System project provided valuable experience in modern application development using Flutter and Firebase, while Business Intelligence demonstrated how data can be transformed into actionable insights through Alteryx and Power BI. High Performance Data Processing introduced scalable approaches for handling large datasets, and Special Topic in Data Engineering reinforced the core principles and technologies that underpin modern data engineering practices.

Overall, this semester significantly strengthened both technical expertise and professional skills while providing practical experience that closely reflects industry expectations within the Data Engineering field.
