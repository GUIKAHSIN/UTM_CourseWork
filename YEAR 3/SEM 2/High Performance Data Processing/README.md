# ⚡ SECP3204 High Performance Data Processing

## 📚 Overview

High Performance Data Processing focuses on techniques and technologies used to process large-scale datasets efficiently. The course introduces performance optimization concepts, scalable processing architectures, and modern data processing frameworks.

---

## 🚀 Featured Project: High Performance Web Crawler

The project involved designing and developing a large-scale web crawler capable of collecting structured data from online sources while evaluating performance optimization strategies.

### Project Objectives

* Collect large volumes of web data
* Evaluate crawler performance
* Compare processing techniques
* Analyze scalability and efficiency
* Support downstream data analytics tasks

### Technologies Explored

* Python
* Pandas
* Polars
* DuckDB
* Web Scraping Libraries
* Performance Benchmarking Techniques

### Key Topics

* Sequential Processing
* Parallel Processing
* Large Dataset Handling
* Memory Optimization
* Performance Evaluation

## 🌱 Reflection

High Performance Data Processing provided valuable insight into the challenges of handling large-scale datasets efficiently. Through the web crawler project, practical experience was gained in collecting and processing large volumes of data while considering performance, scalability, and resource utilization.

The course also introduced modern processing frameworks such as Pandas, Polars, and DuckDB, allowing performance comparisons between different approaches. Understanding how processing techniques affect execution time and memory consumption strengthened my appreciation for efficient data engineering practices and large-scale data processing architectures.
