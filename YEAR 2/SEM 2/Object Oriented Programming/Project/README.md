# ☕ SECJ2154 Object-Oriented Programming

## 📚 Overview

Object-Oriented Programming introduces object-oriented concepts such as classes, objects, inheritance, encapsulation, abstraction, and polymorphism for developing maintainable software systems.

## 📝 Evidence

* Assignment
* Course Project

## 🌱 Reflection

Object-Oriented Programming introduced a new programming language and a different programming paradigm compared to previous courses. Learning concepts such as classes, inheritance, and polymorphism improved my understanding of software design and code reusability. The course project provided practical experience in developing larger and more structured applications using object-oriented principles.
