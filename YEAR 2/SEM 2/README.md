# 📖 Course Highlights

## 📊 Data Analytics Programming

This course introduced data analysis techniques using Python and various data science libraries for data cleaning, transformation, visualization, and interpretation.

### Reflection

Data Analytics Programming was an engaging course that provided valuable exposure to data analysis concepts and techniques. Through this course, I was introduced to Python libraries such as NumPy, Pandas, and Matplotlib for working with datasets. Activities involving data cleaning, transformation, analysis, and visualization demonstrated how raw data can be converted into meaningful insights. The course significantly strengthened my interest in data analytics and provided practical skills that continue to be useful in later data engineering projects.

---

## 🤖 Data Mining

This course introduced techniques for discovering patterns and knowledge from data through various analytical and machine learning approaches.

### Reflection

Data Mining provided valuable exposure to the complete analytical workflow, from data preparation to knowledge discovery. Projects involved data ingestion, preprocessing, normalization, and analysis before applying data mining techniques to identify meaningful patterns within datasets. The course highlighted the importance of data quality and preparation, while also demonstrating how analytical methods can support decision-making and solve real-world problems.

---

## ☕ Object-Oriented Programming

This course focused on object-oriented programming concepts such as classes, objects, inheritance, polymorphism, encapsulation, and abstraction.

### Reflection

Object-Oriented Programming introduced a new programming language and a different approach to software development compared to previous programming courses. Learning concepts such as classes, inheritance, and polymorphism helped improve code organization and maintainability. Developing the course project strengthened programming skills while providing a deeper understanding of how large software systems can be structured using object-oriented principles.

---

## ⚙️ Operating Systems

This course introduced operating system concepts including process management, memory management, scheduling, file systems, and resource allocation.

### Reflection

Operating Systems provided a deeper understanding of how software interacts with computer hardware and system resources. Topics such as process scheduling, memory management, and file systems demonstrated the important role operating systems play in ensuring efficient and reliable computer operation. The laboratory assessments reinforced theoretical concepts through practical activities and experimentation.
