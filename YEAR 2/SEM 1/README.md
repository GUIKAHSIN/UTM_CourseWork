# 📖 Course Highlights

## 📊 Data Structure & Algorithm

This course introduced fundamental data structures and algorithms used to organize and process data efficiently. Topics included arrays, linked lists, stacks, queues, trees, sorting algorithms, and searching techniques.

### Reflection

Data Structure & Algorithm significantly improved my problem-solving and programming abilities. Learning how different data structures and algorithms affect performance helped me understand the importance of writing efficient and scalable programs. The course strengthened my analytical thinking and logical reasoning skills while providing a solid foundation for future software development and technical subjects. Although some algorithm-related concepts were challenging, the knowledge gained became one of the most valuable foundations throughout my computing studies.


---

## 🗄️ Database (WBL)

This course introduced database design, normalization, SQL, and database management concepts.

### Reflection

Database was one of the most important courses for the Data Engineering pathway. The course provided a strong foundation in designing relational databases, writing SQL queries, and understanding how data is stored and managed. More importantly, the knowledge gained from this course was directly applied in the KADA project, where database design played a crucial role in supporting the overall system development process.

---

## 🌐 Network Communications

This course introduced networking concepts, communication protocols, IP addressing, subnetting, and network infrastructure.

### Reflection

Network Communications provided a better understanding of how devices communicate and exchange information within modern computer networks. Topics such as IP addressing, subnetting, and communication protocols introduced the fundamental concepts behind network operations and connectivity.

---

## 🛠 Software Engineering (WBL)

This course focused on software development processes, requirements analysis, and software engineering methodologies.

### Reflection

Software Engineering complemented the KADA project by introducing structured software development practices and documentation techniques. Activities such as requirements analysis, Software Requirements Specification (SRS) preparation, and project planning demonstrated the importance of proper documentation and communication throughout the software development lifecycle.

---

## 💻 System Development Technology

This course introduced practical software development concepts and implementation methodologies.

### Reflection

System Development Technology worked closely alongside Database and Software Engineering through the KADA project. The course provided practical exposure to transforming requirements and designs into a structured system proposal while emphasizing documentation, user manuals, and system planning. Together, these courses provided a complete picture of how software projects are analyzed, designed, documented, and prepared for implementation.

# 📝 Semester Reflection

Year 2 Semester 1 introduced more project-oriented and practical learning experiences compared to previous semesters. Several courses were interconnected through the KADA project, allowing concepts from Database, Software Engineering, and System Development Technology to be applied within a single system development case study. This provided valuable insight into how different areas of computing work together during the development of real-world applications.

Data Structure & Algorithm was one of the most challenging courses during the semester due to the combination of algorithmic thinking and the use of Python libraries such as NumPy, Pandas, and Matplotlib. Although the learning curve was steep, the exposure to data processing and visualization tools later became useful in analytics-related subjects.

Overall, the semester strengthened both technical and project development skills while providing a stronger understanding of software systems, databases, and data processing concepts that would support future studies in Data Engineering.
