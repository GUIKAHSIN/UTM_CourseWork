# 🛠 SECP2513 Software Engineering (WBL)

## 📚 Overview

Software Engineering introduces software development methodologies, requirements analysis, project planning, software quality assurance, and software documentation.

## 🚀 Featured Project: KADA-eServe

KADA-eServe is a cooperative management system developed as part of the course project. The project involved identifying user requirements, analysing business processes, and producing software engineering documentation to support system development.

## 📝 Evidence

- Academic Writing
- Software Requirements Specification (SRS)
- KADA Project Documentation

## 🌱 Reflection

Software Engineering introduced the importance of structured software development processes and proper documentation. Through the KADA project, experience was gained in requirements gathering, stakeholder analysis, project planning, and documentation preparation. The course demonstrated that successful software development requires not only technical skills but also effective communication, planning, and collaboration.
