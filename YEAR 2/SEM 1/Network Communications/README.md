# 🌐 SECR2043 Network Communications

## 📚 Overview

Network Communications introduces networking concepts including communication protocols, IP addressing, routing, subnetting, and network infrastructure.

## 📝 Evidence

* Lab Activities
* Assignment
* Group Project

## 🌱 Reflection

Network Communications provided a better understanding of how devices communicate and exchange information within modern computer networks. Topics such as IP addressing, routing, and subnetting introduced the fundamental concepts behind network connectivity and communication. The course provided valuable knowledge on how networked systems operate and support digital communication.
