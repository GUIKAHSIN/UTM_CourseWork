Database was one of the most important courses related to the Data Engineering pathway. Learning how to design relational databases, create entity relationship diagrams, normalize data, and write SQL queries provided a strong foundation in data management. The concepts learned were directly applied in the KADA project, where database design played a crucial role in supporting the overall system development process.
